## Async Operations Pattern
Asynchronous endpoints follow this pattern:
1. Client sends request with callback URL and credentials
2. Server immediately returns a `callbackId`
3. Server processes the request in background
4. When complete, server POSTs result to the provided callback URL
5. Callback includes the `callbackId` for correlation and a `payload` with the actual result
